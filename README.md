# To-Do-List App
A simple and responsive To-Do List built with HTML, CSS, Bootstrap, and JavaScript.

## Features

-  Add tasks
-  Edit tasks
-  Delete tasks
-  Mark tasks as completed
-  Display today's date
-  Motivational message
-  Responsive design

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
